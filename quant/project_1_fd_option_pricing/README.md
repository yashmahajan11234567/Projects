Finite Difference PDE Solver for Option Pricing

This project implements finite-difference solvers for European and American options under the Black–Scholes model.

Features:
- Crank–Nicolson scheme for European options
- PSOR solver for American options (free-boundary problem)
- Greeks computation from PDE grid
- Benchmarking vs Monte Carlo

This project aligns with quantitative finance and fintech pricing roles.
