\# DMW – Data Mining \& Warehousing Project



This project analyzes the Instacart grocery dataset using Python and Power BI.



\## Contents

\- `instacart\_analysis.py` – data analysis and feature engineering

\- `prepare\_exports\_for\_powerbi.py` – prepares data for visualization

\- `Miniproject.pbix` – Power BI dashboard
